## Installation

To install, use `go get` using go modules:

```
go get github.com/edermanoel94/rest-go@latest
```

## Example

To install and build:

```
cd examples
```

And use go modules for install packages:

```
go mod tidy 
```

Then, to run:

```
go build
```
